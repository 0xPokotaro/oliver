# Glossary: 用語集

## Syntropy（シントロピー）

AIエージェントが人間の代わりに「推論」し、Web上のあらゆるサービスと「決済」を行うための、UIを持たない（Headless）商取引インフラ。

## x402

HTTP 402 Payment Required ステータスコードを拡張した、決済ゲートウェイプロトコル。

## Orchestrator（オーケストレーター）

Layer 1の役割を担う、ユーザーの意図（Intent）を理解し、指揮する脳。音声I/O、文脈理解、タスクのルーティングを担当。

## Facilitator（ファシリテーター）

ガスレス決済を実現するためのリレイヤー。署名を検証し、オンチェーン決済を実行する。
