# ガジェット操作仕様

このドキュメントでは、Oliverシステムのガジェット（物理デバイス）の操作仕様について説明します。

## 概要

Oliverガジェットは、e-Paperディスプレイ、音声入出力、コントローラーを統合したデバイスです。ユーザーはコントローラーと音声入力を組み合わせて、AIエージェントと対話することができます。
電源を入れると、ディスプレイが起動し表示が開始されます。

## ディスプレイ表示

e-Paperディスプレイには以下の情報が表示されます：

| 項目 | 内容 |
|---|---|
| **設定状態** | WiFi接続状態 |
| **ユーザー情報** | ウォレットアドレス、QRコード、JPYC残高 |
| **現在日時** | 現在の日時 |

## コントローラー操作

### 音声入力の制御

| ボタン | 操作 | 動作 |
|---|---|---|
| **Xボタン** | 押下（音声入力オフ時） | 音声入力がオンになる |
| **Xボタン** | 押下（音声入力オン時） | 音声入力がオフになり、収録された音声がAPIを通してAIに送信される |

### その他のボタン

| ボタン | 機能 |
|---|---|
| **十字キー** | メニューナビゲーション |
| **Aボタン** | 決定 |
| **Bボタン** | キャンセル |
| **L/Rボタン** | 補助操作 |
| **Start/Selectボタン** | メニュー・設定 |

## 操作フロー

0. **電源オン**: 電源を入れるとディスプレイが起動し、表示が開始されます
1. **待機状態**: ディスプレイにWiFi接続状態、ユーザー情報、現在日時が表示されます
2. **音声入力開始**: Xボタンを押すと音声入力がオンになります
3. **音声入力終了**: 再度Xボタンを押すと音声入力がオフになり、収録された音声がAPIを通してAIエージェントに送信されます
