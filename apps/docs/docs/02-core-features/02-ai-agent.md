# AI Agent

Oliverは、AIの構造を **「思考する人格」**と**「実務を行う能力」** に明確に分離することで、柔軟性と堅牢なセキュリティを両立しています。

## 2層アーキテクチャ

**「人格」と「能力」の分離**

OliverのAIエージェントは、2つのレイヤーで構成されています。

## 🗣 Layer 1: オーケストレーター（人格）

**役割:** ユーザーとの対話を通じて文脈を理解する「コンシェルジュ」

"いつものやつ" という曖昧な言葉を理解し、生活習慣に基づいてタスクを整理します。

### 特徴

- 思考と対話に特化
- 文脈理解と意図の解釈
- タスクの優先順位付け

### セキュリティ

⚠️ ウォレットの秘密鍵には一切アクセス不可

---

## ⚡ Layer 2: スペシャリスト（能力）

**役割:** 実際のタスクを厳格に執行する「実務家」

x402プロトコルを解釈し、署名を生成して決済を完了させます。

### 特徴

- 購買モジュール
- 予約モジュール
- 労働力調達モジュール

など、特定の「能力」に特化したモジュールとして動作します。

### セキュリティ

✅ 署名生成と決済実行を担当
