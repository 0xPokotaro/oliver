# AI Agent

## 思考と実務の分離、設計された安全

Oliverは、AIの構造を **「思考する人格」**と**「実務を行う能力」** に明確に分離することで、柔軟性と堅牢なセキュリティを両立しています。

**AIエージェントは、2つの独立したレイヤーの相互作用によって動作します。**

## AI Agentが実現する「2層アーキテクチャ」

### 1. 【人格層】オーケストレーター

ユーザーとの対話を通じて文脈を理解する「コンシェルジュ」としての役割を担います。"いつものやつ" という曖昧な言葉を理解し、生活習慣に基づいてタスクを整理します。

- **思考と対話**: 文脈理解と意図の解釈に特化し、ユーザーの曖昧な指示を具体的なタスクへと変換します。
- **タスク管理**: 複数のタスクの優先順位付けを行い、適切な順序で処理を実行します。
- **セキュリティ（秘密鍵の分離）**: ウォレットの秘密鍵には一切アクセスできない設計となっており、資産流出のリスクを根本から排除します。

### 2. 【能力層】スペシャリスト

実際のタスクを厳格に執行する「実務家」としての役割を担います。x402プロトコルを解釈し、署名を生成して決済を完了させます。

- **モジュール化された能力**: 購買、予約、労働力調達など、特定のタスクに特化したモジュールとして動作します。
- **決済の実行**: 署名生成と決済実行を担当し、プロトコルに従って正確に取引を完結させます。
