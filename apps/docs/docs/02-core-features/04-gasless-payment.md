# ガスレス決済
**ブロックチェーンの複雑さを排除**

ユーザーやAIエージェントが、決済のために暗号資産（ETHやMATIC）を管理する必要はありません。Oliverはブロックチェーンの複雑さを完全に隠蔽します。

*   **署名ベースの決済:** 決済に必要なのは「署名」だけです。EIP-2612 (Permit) により、オンチェーン送金ではなくオフチェーン署名で承認を行います。
*   **ファシリテーター・モデル:** ガス代は「Facilitator（中継者）」が代行して支払います。
*   **ステーブルコイン・ネイティブ:** ユーザーはUSDCやJPYCなどのステーブルコインさえ持っていれば、すぐに生活必需品を購入できます。
