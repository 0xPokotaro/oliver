# AI Payment

## ガスレス、かつ安全な決済インフラ

A2A Commerceは、AIエージェントが「迷わず、安全に」取引を完結できるインフラを提供します。

**AIに自由を与えつつ、人間が設定した境界線を越えさせない「設計された安全」を提供します。**

## AI Paymentが提供する「2つの革新」

### 1. 【ユーザー視点】ガスレス決済

AIエージェントにガス代（ETHやMATIC等）を管理させる必要はありません。ブロックチェーン特有の複雑さを排除し、スムーズな取引を実現します。

- **署名のみで完結**: EIP-2612（Permit）を活用。オンチェーン送金ではなく、オフチェーン署名だけで決済を承認します。
- **ファシリテーター・モデル**: 手数料は中継者が代行。ユーザーはUSDCやJPYCなどのステーブルコインさえ持っていれば、即座に経済活動を開始できます。

### 2. 【セキュリティ視点】オンチェーンでの行動制御

AIが自律的に経済活動を行う上で、人間が安心して任せられるための制御機構を提供します。

- **インテントベースの署名 (EIP-712)**: 「何を・いくらで・いつ買うか」が明記された構造化データにのみ署名。不透明なハッシュへの署名を排除し、取引の透明性を確保します。
- **スマートポリシー (ERC-7579)**: 「月次予算上限」や「過去の平均購入額の+10%以内」といった厳格なポリシーを設定。AIの暴走や逸脱した取引を自動的にブロックします。
