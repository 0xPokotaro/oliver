# セキュリティ
**設計された安全性**

生活の一部をAIに任せるためには、盤石な安全性が必要です。OliverはオンチェーンレベルでAIの行動を制御します。

*   **インテントベースの署名 (EIP-712):** 「何を・いくらで・いつ買うか」が明記された構造化データに対してのみ署名します。不透明なハッシュへの署名は行いません。
*   **スマートポリシー (ERC-7579):** AIに無制限の権限は与えられません。「過去の平均購入額の+10%以内」「月次予算上限」といった厳格なポリシーを設定し、逸脱する取引は自動的にブロックされます。
