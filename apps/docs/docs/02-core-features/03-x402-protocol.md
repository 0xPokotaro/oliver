# x402プロトコル
**摩擦のない「URLベース」の商取引**

Oliverは、人間用のWebサイト（GUI）を経由しません。
Webの標準ステータスコードである **HTTP 402 (Payment Required)** を活用し、**「URLにアクセスするだけ」** で決済が完了する世界を実現します。

*   **ログイン不要:** IDやパスワードの管理は不要です。ウォレットによる署名が認証となります。
*   **入力フォーム不要:** 住所やカード情報の入力フォームは存在しません。必要な情報はプロトコル経由で安全に渡されます。
*   **高い汎用性:** 既存のAPIにミドルウェアを導入するだけで、あらゆるサービスがOliver対応になります。
